## A Component Measuring Program Based on CV
- Date: 2018-10-16
- Version: 1.0.0
- Abstract: Program measures shape parameters of component based on input image

---

### Dependency
```
- OpenCV 3.0
```


### Project structure

```
Component-Measure
|- bin/ (executable)
|- image/ (input images)
|- include/  (header file)
|- src/  (source files)
|- README.md
|- CMakelists.txt
```

---

### Compile and Run

```
# compile
cmake .
make

# runtest
```
